# Arctic Wolf Dispatch App

24/7 Kenai Peninsula towing and roadside assistance dispatch platform.

## Features

- 📨 Quick SMS dispatch form
- 💬 Pre-formatted service requests
- 📋 Copy-to-clipboard fallback for desktop
- 🎨 Dark arctic theme with Tailwind CSS
- 📱 Fully responsive design
- 🛻 Multiple service types (towing, winch-outs, roadside assistance)

## Quick Start

### Prerequisites
- Node.js 16+ and npm installed

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tka907480-coder/Arctic-wolf-dispatch.git
cd Arctic-wolf-dispatch
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The app will open at `http://localhost:3000`

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Runs the test suite

## Technologies

- React 18
- Tailwind CSS 3
- Vite (via create-react-app)
